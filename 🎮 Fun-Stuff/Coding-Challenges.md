## ğŸ’¡ MrDib's Challenge Strategy

</div>

### Effective Practice Routine

**Daily Routine (30-60 mins)**

1. **Warm-up**: 1 easy problem (5-10 mins)
2. **Main Focus**: 1-2 medium problems (20-40 mins)
3. **Stretch Goal**: 1 hard problem (attempt, even if incomplete)

**Weekly Routine**

- **Monday-Friday**: Daily problems
- **Saturday**: Participate in a contest (LeetCode/Codeforces)
- **Sunday**: Review contest problems, learn new concepts

**Skill Development**

1. **Week 1-2**: Arrays & Strings (Two Pointers, Sliding Window)
2. **Week 3-4**: Hash Maps & Sets
3. **Week 5-6**: Linked Lists, Stacks, Queues
4. **Week 7-8**: Trees & Binary Search Trees
5. **Week 9-10**: Graphs (BFS, DFS)
6. **Week 11-12**: Dynamic Programming Basics
7. **Week 13+**: Advanced topics (Backtracking, Greedy, Heap, Trie)
